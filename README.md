# Опис проєкту: реалізація операцій над множинами на деревах

Проєкт присвячений реалізації абстрактного типу даних «множина» (Set) із повним набором базових операцій, використовуючи різні структури пошуку на деревах. Кожен реалізує свою структуру даних і забезпечує однаковий інтерфейс роботи з множинами (додавання, видалення, перевірка належності тощо), щоб можна було порівнювати коректність і продуктивність.
